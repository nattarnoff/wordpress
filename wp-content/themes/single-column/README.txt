An accessible single-column WordPress theme with block editor support, a full-screen hero, and responsive navigation.

== Description ==

Single Column is a lightweight, accessibility-focused WordPress theme.

Theme structure includes:

- style.css (required)
- templates/index.html (required for block template fallback)
- functions.php
- theme.json
- classic PHP templates for header/footer/index/singular/page/front page

== Installation ==

1. Upload the `single-column` folder to `/wp-content/themes/`.
2. In WordPress admin, open **Appearance → Themes**.
3. Activate **Single Column**.
